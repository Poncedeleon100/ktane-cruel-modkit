On the Subject of The Cruel Modkit

It is literally do or die; and dying seems likely.

See Appendix A for indicator identification reference.
See Appendix B for battery identification reference.
See Appendix C for port identification reference.

  • The module consists of a display with module components, a “❖” button, a “◆” button, multiple smaller buttons and several containers for a wire panel, a button panel, a set of LEDs, a symbol keypad, an alphanumeric keypad, a piano, a set of arrow buttons, and a panel for two bulbs.
  • There are also widgets embedded in the module. These include a meter, two resistors, an identity card, a timer, a word display, a number display, and an LED that transmits a message in Morse code*.
  • In order to disarm the module, choose the right components for it and then solve it using the corresponding instructions in the pages below.
  • Interacting with module components when the current component selection is not the correct one will cause a strike.
  • Use the arrow buttons near the top display to cycle between the possible module components and press the “◆” button to toggle the corresponding component.

To avoid confusion, zeros in this manual are represented by the character Ø.

Refer to the steps below to determine what components to include.†

  1. Split the serial number into three groups of two characters.
  2. Take the individual groups and multiply their characters together using their Base36 values.
  3. Take the sum of each of the three products, modulo 256.
  4. Convert the number to binary.
  5. With the first digit of the binary number representing the wires component, activate each component that has a 1.

*Go here for Morse Code reference.

On the Sub-Subject of Timer Timings

“Okay, give a read on the module.” “There’s nothing.” “Okay… 6/3.” “No, there’s actually nothing.” “1/8.” “No… THERE IS NOTHING ON THE MODULE!” “9/2.”

Press the "❖" symbol to activate the module. The timer on the module will begin displaying random digits from ØØØØØ-99999, changing every second. The number display will also give a number. Refer to the table below to find out what number to submit. The left-most digit represents A, and the right-most digit represents B.

Number Display Rule
Ø A + B = a prime number*
1 A > amount of lit indicators, B ≤ amount of unlit indicators
2 A / B = a whole number
3 A and B = a multiple of the module count, excluding needies**
4 The digital root of A + B is odd

8 A + B ≥ sum of S# digits modulo 18
9 A and B = the amount of lit or unlit indicators.

When the rule applies to the digits, press the "❖" symbol again to solve the module. Incorrectly pressing the "❖" symbol will result in a strike.

*Ø in this module is considered a composite number.

**If the module count is over 1ØØ, press the "❖" symbol when A and B share parity with the last digit of the serial number.

On the Sub-Subject of Quaternary Binary

This module is 1D Maze, but without the Maze part… So just 1D?

Note: If your module's components aren't exactly [Bulb], you are looking at the wrong section of this manual!

Press the "❖" symbol to activate the module, unlocking the I/O keys. The bulbs will now flash a series of binary digits. The sequence has eight flashes, with a long pause to separate them. How the module generates this sequence is provided:

  • The module will choose a random row and column in the table below.
  • The row will be the maze in its unchanged form.
  • Three random digits within the row will differ.
  • The row and column intersection will be the leftmost position in the maze, shifting the maze left until it is such, wrapping around the numbers.
  • The digits are then converted to binary and displayed using the bulbs.

Once you have determined what digits are different and what the initial row looked like, hold either button for three seconds to enter submission. The bulbs will turn off to accommodate this.

To solve the module, convert the replaced digits into binary, concatenate them in the order they appear in the row, and submit the sequence using the buttons. The module will only check your answer after submitting all eight digits. The module will grant a strike if any of the digits are incorrect.

On the Sub-Subject of Simon Smarts

“TOP 10 9232092 IQ PLAYS DONE BY SIMON!!! (NUMBER 8 WILL SHOCK YOU)” -Sameone

Note: If your module’s components aren’t exactly [Arrows], you are looking at the wrong section of this manual!

The module has four colors, forming a diamond (red, yellow, green, and blue). Between the diamond colors are four circle colors (magenta, cyan, white, and black). There will be five stages, each adding to the previous sequence of flashes. The stage number can be tracked with the number display. Press the “❖” to start the button flashing.

  • First, decide which colors are 1s and which are 0s by locating the position of the red color on the diamond:
    • If it's on the left, every circle color is a one, and the rest are 0s.
    • If it's on the right, every diamond color is one, and the rest are 0s.
    • If it's on the top, red, white, and black are 0s, and every other color is a 1.
    • If it's on the bottom, every color that isn't White, Black, Cyan, and Red are 1s. The rest are 0s.
  • Press every color in the following sequence after each stage:
    1. The first color that was flashed
    2. The color(s) received from the first color XOR the second color.
    3. The color(s) received from the colors pressed during Stage 2 XNOR the first color that was flashed.
    4. The color(s) received from the third color that was flashed NAND the color pressed in Stage 1.
    5. The color(s) received from (Stage 2 AND Stage 4) XOR (Stage 1 NOR Stage 3).

Toggle on/off every color that applies to the current stage, then submit using the middle button. Repeat this for every individual stage to advance the module. Finishing the fifth stage will solve the module. If the submission isn't complete, the flash cycle restarts. Pressing the wrong color will cause a strike.

On the Sub-Subject of Cruel LED Pattern

Look at those pretty lights… On second thought, best to stay focused.

Note: If your module’s components aren’t exactly [LED], you are looking at the wrong section of this manual!

To disarm the module, locate the LED pattern present in the module in the grid on the next page and identify the sections it belongs to.

The LED pattern must be in a horizontal/vertical line of 8 colors, wraparound included on the grid, forwards or backwards. The 16x14 grid is separated into different sections horizontally and vertically. Horizontally, each section takes up 4 columns and is numbered Ø-3. Vertically, each section takes up 2 rows and is numbered Ø-6.

If a pattern is part of more than one section, use the number of the section that contains the highest amount of the pattern compared to every other section the pattern is part of, taking the lowest number if any sections are tied. Otherwise, use the section number that contains the entire pattern. Find the correct section numbers from the horizontal and vertical sections.

Once the correct section numbers have been found, press the “❖” button when the last seconds digit on the countdown timer matches the sum of the two section numbers.
