Note: If your module’s components aren’t exactly [COMPONENTS], you are looking at the wrong section of this manual!

  • Submodules should be ordered numerically using their respective puzzle IDs. Check the spreadsheet to see this order.
  • css/Modules/The Cruel Modkit.css contains the stylesheet used for all of the manuals.
    • Prefix class names with the submodule name, using “-” for spaces.
    • Use the same order as above.
  • js/Modules/The Cruel Modkit.js contains the scripts used for all of the manuals.
    • Prefix variable names with the submodule name.
    • Use the same order as above.
  • Keep the following in mind when writing manuals:
    • Use “these” type of quote marks to quote text in paragraphs and ‘these’ single quotes for apostrophes or single quotes.
    • Replace “serial number” with “SN”.
    • Replace ANY zeroes with the Ø symbol, unless it is the page number or an ordered list.
